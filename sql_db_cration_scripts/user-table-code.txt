To setup the Database follow the steps here https://www.youtube.com/watch?v=N676lCQq4n8


1. Open xampp
2. Start Apache and MySQL
3. On MySQOL click on Admin
4. Create a database
5. Create the table user 
- you can do it by clicking the database, click new, add the columns
OR
- open the SQL tab and run the following code


CREATE TABLE IF NOT EXISTS `user` ( 
  `username` varchar(200) NOT NULL,  
  `password` varchar(200) NOT NULL,
  `email` varchar(50) NOT NULL, 
  `account_type` enum('Manager','Developer') NOT NULL, 
  PRIMARY KEY (`username`)  
) ENGINE=InnoDB DEFAULT CHARSET=latin1;
